# 🎓 Student Management System (Python)

A simple command-line **Student Management System** built using **Python**. This application allows users to manage student records by adding, viewing, searching, updating, and deleting student information. All records are stored in a CSV file, ensuring the data is saved even after the program is closed.

---

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student Record
- 💾 Automatic CSV File Creation
- 📂 Persistent Data Storage
- ❌ Input Validation

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Student-Management-System.git
```

2. Open the project folder

```bash
cd Student-Management-System
```

3. Run the application

```bash
python main.py
```

> **Note:** If `students.csv` does not exist, it will be created automatically when the program starts.
